Из чего будет состоять SmartBot?

Вторник, 18 Июня 2013

Ранее я рассказал, какие цели преследуются при разработке автономного робота “SmartBot”. Для реализации задачи необходимо выбрать основные компоненты робота.

Основные компоненты робота
На данном этапе разработки будет составлен перечень основных компонентов, из которых будет состоять робот.

1. Платформа робота:
Существует множество готовых решений, для различного класса роботизированных устройств. Так как по периметру платформы устанавливается серия ультразвуковых датчиков, для удобства размещения и улучшения технических характеристик, платформа выбирается круглой формы рис. 1.

2. Датчики:
Основным датчиком робота, будет серия ультразвуковых датчиков “Parallax Ping))” рис. 2(а). Для установки датчиков будет использоваться подставка “Acrylic Ping))) Stand with Hardware” фирмы “Parallax” рис. 2(б).

3. Микроконтроллер:
Для управления роботом будет использоваться плата Arduino Mega 2560 и BeagalBoard рис. 3 (а,б).

4. Мотор и колеса:
Был выбран мотор-редуктор Motor Mount and Wheel Kit фирмы Parrallax, в комплект входит два мотор-редуктора с колесами рис. 4(а). Так же помимо мотор-редукторов будет установлено два поворотных колеса Caster Wheel Kit фирмы Parallax рис. 4(б).

5. Драйвер управления мотором
Для управления мотор-редуктором был выбран драйвер управления HB-25 Motor Controller фирмы Parallax рис. 5.

6. Элемент питания
Для автономного передвижения потребуется источник питания, способный обеспечить робота энергией, необходимой для выполнения работы. В качестве такого источника будет использоваться два аккумулятора фирмы Zippy рис. 6.
